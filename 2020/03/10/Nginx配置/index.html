Nginx配置

查看Nginx配置

注意上面的几个配置：

  • –conf-path : /usr/local/etc/nginx/nginx.conf 配置文件
  • –http-log-path：访问日志
  • –error-log-path：错误日志

nginx.conf配置文件

前面我们知道了，/usr/local/etc/nginx/nginx.conf 是nginx服务运行过程中的具体配置文件
这一部分，我们将通过剖析nginx.conf文件，弄清下面几点：

  1. 设置端口？
  2. 开启日志？修改日志存储路径？
  3. 配置反向代理映射规则？

文件内部结构

nginx.conf文件内部结构：

  1. worker_processes
  2. error_log
  3. events
  4. http
    1. access_log
    2. server
      1. listen
      2. server_name
      3. access_log
      4. location
        注：https时，上面的server配置略有不同。
